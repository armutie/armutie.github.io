Autonomous RC Aircraft

From manual flight to GPS waypoint navigation

The Vision

Building an autonomous fixed-wing aircraft from scratch. Starting with manual flight, then incrementally adding autonomy — stabilization, auto takeoff/landing, and eventually GPS waypoint navigation.

Roadmap

Phase 1 Current

Get It Flying

Build the airframe, install electronics, achieve stable manual flight.

  • Design and build fuselage + wings
  • Install motor, ESC, servos, receiver
  • First manual flight test
Phase 2

Auto-Stabilization

Add an ESP32 with IMU for PID-based flight stabilization.

  • Integrate ESP32 + MPU6050
  • Implement PID control for pitch/roll
  • Test stability assist mode
Phase 3 Stretch Goal

Auto Takeoff & Landing

Altitude hold and automated takeoff/landing sequences.

Phase 4 Stretch Goal

GPS Navigation

Autonomous flight between GPS waypoints.
